#  Gridless Tech: Solar Bottle Lights Monitoring System

An IoT-Based Solar Bottle Lights with Energy Consumption Monitoring using MQTT, INA219 sensor, and a real-time web dashboard for remote control and data visualization.

---

## 📌 Project Overview
This system is designed to provide sustainable lighting using solar energy while allowing users to monitor electrical parameters such as voltage, current, power, and energy consumption in real-time.

---

## ⚙️ Technologies Used
- Arduino / ESP Microcontroller
- MQTT Protocol
- INA219 Current Sensor
- HTML, CSS, JavaScript (Web Dashboard)

---

## 📡 Features
- 🔆 Solar-powered lighting system
- 📊 Real-time monitoring (Voltage, Current, Power)
- 🎛 Remote light control via web dashboard
- 🔋 Energy consumption tracking
- 🌐 MQTT-based communication

## 🚀 How to Use

1. Download or clone the repository
2. Upload Arduino code to your microcontroller
3. Connect sensors (INA219)
4. Configure MQTT settings in the code
5. Open `index.html` in your browser

---

## ⚠️ Notes
- Real-time functionality requires an active MQTT broker
- Hardware setup is needed for full system operation
